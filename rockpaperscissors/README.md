# Rock Paper Scissors Game (Python)

A simple command-line **Rock–Paper–Scissors** game written in Python where the player competes against the computer.

## Features
- Interactive command-line gameplay  
- Player name input  
- Clear game rules displayed at the start  
- Random computer choice  
- Input validation for user choices  
- Option to replay the game multiple times  

## Game Rules
- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Same choice results in a draw  

## Technologies Used
- Python  
- random module  

## How to Run
1. Make sure Python is installed on your system  
2. Save the file (for example: `rps.py`)  
3. Run the program using:
```bash
python rps.py
```
